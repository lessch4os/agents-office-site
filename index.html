agents-office

Real-time observability dashboard for AI coding agent sessions. Each running Claude Code, OpenCode, or Antigravity session shows up as an animated agent in a web-based office.

GitHub npm License

Architecture

┌──────────────────┐ Unix socket ┌──────────────────┐ │ CC Hook │ ──────────────────► │ │ │ (hook-shim) │ JSON/event stream │ Bun Daemon │ └──────────────────┘ │ (daemon/src/) │ │ │ ┌──────────────────┐ │ ┌─ Hook Socket │ │ CC Sessions │ JSONL watch │ ├─ JSONL Watch │ │ (JSONL) │ ──────────────────► │ ├─ Reducer │ └──────────────────┘ │ ├─ State │ │ ├─ WebSocket │ ┌──────────────────┐ JSONL watch │ ├─ SSE Watcher │ │ Antigravity │ ──────────────────► │ └─ Plugin │ │ Sessions │ └────────┬─────────┘ └──────────────────┘ │ WebSocket │ ┌──────────────────┐ Unix socket │ │ OpenCode Plugin │ ──────────────────► │ │ (opencode-plugin)│ │ └──────────────────┘ │ ▼ ┌──────────────────┐ ┌──────────────────┐ │ Forwarder │ WebSocket (TLS) │ Web Frontend │ │ (client machine) │ ──────────► /hook │ (web/src/) │ │ Unix socket → WS │ │ React + PixiJS │ └──────────────────┘ └──────────────────┘

Quick start

Pre-built binary (no dependencies)

./daemon/agents-office --port 8080

npm (requires Bun)

npx @lessch4os/agents-office --port 8080

From source

bun install
bun run daemon/src/main.ts --port 8080

Homebrew

brew tap lessch4os/agents-office
brew install agents-office
brew services start agents-office

Open http://localhost:8080 in your browser.

Supported sources

  • Claude Code — via hook shim + JSONL watcher
  • OpenCode — via plugin or native SSE watcher
  • Antigravity — via JSONL brain file watcher
